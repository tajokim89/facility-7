## 2. 반영 원칙

- **1회차**: 세계관 요소를 "평범한 직장의 용어/규칙"처럼 자연스럽게 노출
- **2회차**: 같은 요소가 공포로 재해석되는 오버라이드 추가
- **requiredEndings/requiredFlags**: 특정 엔딩/선택 경험 후에만 드러나는 심층 세계관 (스키마에 이미 있으나 미사용 중인 기능)
- 기존 노드 텍스트 수정 + 오버라이드 추가 위주, 신규 노드는 최소화

## 4-1. 감정 자원 체계 게임 시스템 반영

### 현재 vs 세계관 갭

| 항목 | 현재 게임 | 세계관 (감정_자원_체계.md) |
|------|----------|--------------------------|
| 수치 이름 | `remainingEmotion` (R.E. 0~100) | 분(m) — 느낄 수 있는 시간 |
| 감정 종류 | "감정 유형: 권태" 텍스트만 | 등급제 SSS/S/A/B/F + 색상 + 환율 |
| 감소 원인 | emotionDelta 숫자 | 추출당할 때 수명이 소진되는 개념 |
| 0 도달 시 | ending_empty 강제 이동 | 공허자 전환 — 자아 소멸, 시설 부품화 |
| 경제 시스템 | 없음 | 분(m) 화폐, 등급별 환율, 암시장 |

### 감정 등급표 (세계관 원문)

| 등급 | 이름 | 내용 | 환율 |
|------|------|------|------|
| SSS | The Genesis | 희생적 사랑, 완벽한 이타심 | HU-01 연료 (환율 없음) |
| S | The Ambrosia | 첫 키스, 승리의 환희 | 1ml = 1,440m (24시간) |
| A | Clear Blue | 맑은 날의 평온, 사소한 유머 | 1ml = 60m |
| B | Bitter Sweet | 아련한 향수, 이별의 슬픔 | 암시장 고가 (일반 유통 금지) |
| F | The Dregs | 극심한 공포, 자기혐오, 살의 | -1,000m (위험 수당 별도) |

### 필요한 시스템 변경

#### `game/src/data/schema.ts`

```typescript
export type EmotionGrade = 'SSS' | 'S' | 'A' | 'B' | 'F';

// SceneNode에 추가

```

#### `game/src/engine/GameEngine.ts`
- `remainingEmotion` 내부 로직 유지 (0~100)
- UI 표시용 getter 추가: `getMinutesDisplay()` — 분(m) 환산값 반환
  - `remainingEmotion 100` → `"1,440m"` / `50` → `"720m"` / `0` → `"0m"`

```typescript
getMinutesDisplay(): string {
  const minutes = Math.round(this.remainingEmotion * 14.4);
  return `${minutes.toLocaleString()}m`;
}
```

#### `game/src/components/EmotionGauge.tsx`
```tsx

// 변경 후

```
